# Quick start

## Install the command line tools

` $ npm install --g expo-cli `

## Clone this repository
` $ git clone https://github.com/5626pedro/insta-cat-app `

## Go into the repository
` $ cd insta-cat-app `


## Start the local development server of Expo CLI

`$ expo start`


# iOS Simulator
